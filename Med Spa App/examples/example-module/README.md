# Example Module

A reference marketplace module that demonstrates the Baseplate module format.

## What It Does

Provides a simple dashboard page showing appointment, revenue, and patient summaries. This module is for demonstration purposes — it shows how to:

1. Structure a `manifest.json`
2. Create a UI page that renders in the portal
3. Create an API endpoint that returns data
4. Declare permissions and dependencies

## Installation

This module is included in the Baseplate monorepo under `examples/`. In production, clinics would install it from the marketplace browse page.

## Manifest

See `manifest.json` for the complete module declaration.

## Pricing

This example is priced at $29/month (subscription). In a real marketplace, you'd set your own price based on the value your module provides.
